High transverse resolution



It is only possible to run 2D probe simulations, because for simulation parameters
$$
L_x=100 um, L_y=L_z=80 um, \Delta x=10 nm, \Delta y=\Delta z=50 nm
$$
It will require at least 1335 GB memory in a single time step.
For 2D simulations, it is 0.83 GB.
